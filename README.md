# LangChain Agent

This is the first langchain agent. Will require node 20 or above

## Installation
1. run `npm i`
2. add `.env` file to the root project with a open ai api key (OPENAI_API_KEY) and a tavily key (TAVILY_API_KEY)

## Running Instructions
1. `node indexjs`
